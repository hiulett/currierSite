# Operational Core: LogySaaS Admin Dashboard

Redesign the Admin Dashboard to reflect the "Technical Elegance" aesthetic. The goal is to provide a "Mission Control" feel with high data density but clean organization.

**DESIGN SYSTEM (REQUIRED):**
### Visual Theme & Atmosphere
Technical Elegance. Deep Space Slate (#0f172a) background for sidebar and header. High contrast with Cloud White (#f1f5f9) text. Subtle glow effects for active status indicators.

### Color Palette
- **Deep Space Slate (#0f172a):** Main background and navigation.
- **Logi-Blue (#3b7ddd):** Primary action buttons and active icons.
- **Emerald Pulse (#10b981):** Successful delivery / Paid status.
- **Amber Warning (#f59e0b):** In-transit / Pending issues.
- **Surface Navy-Slate (#1e293b):** Card backgrounds.

### Typography
- **Primary Font:** Plus Jakarta Sans.
- **KPI Metrics:** Semi-bold, large tracking (-0.05em).

**Page Structure:**
1. **Sidebar:** Retractable, icon-heavy with clear labels. Dark theme.
2. **Top Bar:** Breadcrumbs, global search, and user profile with role badge.
3. **KPI Row:** 4 large cards showing "Today's Packages", "Pending Yappy Approvals", "Active Shipments", and "Monthly Revenue".
4. **Active Inventory Table:** A data-dense table with custom status badges and quick action buttons.
5. **Real-time Map/Activity Feed:** A small section showing the latest movement in the warehouse.
